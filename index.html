RDFIA - Master DAC - Sorbonne University

This page contains the coding sessions done for the RDFIA course.

The course is organized by Prof. Matthieu Cord. The teacher assistants, to whom only you must send the results of coding sessions, are Arthur Douillard (arthur.douillard@lip6.fr), Alexandre Rame (alexandre.rame@lip6.fr) and Asya Grechka (asya.grechka@lip6.fr).

To simplify our job, send all emails to ALL teacher assistants and the email object must be [RDFIA][Homework-*] with the * indicating the homework identifier (1, 2, or 3). If it's not related to a homework, please use [RDFIA][MISC].

Send your homework in ENGLISH please!

All coding sessions will be done in Python3 and we'll use several libraries. Here are some useful links if you're rusty:
