This page contains the coding sessions done for the RDFIA course.

The course is organized by Prof. Matthieu Cord. The teacher assistants, to whom only you must send the results of coding sessions, are Arthur Douillard (arthur.douillard@lip6.fr), Alexandre Rame (alexandre.rame@lip6.fr) and Asya Grechka (asya.grechka@lip6.fr).

To simplify our job, send all emails to ALL teacher assistants and the email object must be [RDFIA][*] with the * indicating the homework identifier (I, II, or III). If it's not related to a homework, please use [RDFIA][MISC].
